Tools

GitHub

GitHub is a web-based Git or version control repository and Internet hosting service. This allowed out team to commit and update changes to the website in real time. We could work remotley, but still be able to manage the source code.

HTML

HTML is a computer language devised to allow website creation. We used it to put text, images, links and other elements on our website. It was also used to position elements in various positions on the webpages.

Bootstrap

HTML, CSS, and JavaScript framework for developing responsive websites for mobile and desktop use. Many of the styles were built in to Bootstrap and this meant less code would have to be written for styling some of the elements.

Slack

Advanced communication for groups and teams used to collaborate on projects. Our team used Slack to split up responsiblities and troubleshoot any problems that we were having.
